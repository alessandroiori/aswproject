# asw-000-ciao-mondo

Questo progetto contiene una sola applicazione **ciao-mondo**, 
che non è un'applicazione distribuita. 

## Build  

Per la costruzione dell'applicazione, vedere le istruzioni 
descritta nella sezione [projects/](../). 

## Componenti eseguibili 

L'applicazione **ciao-mondo** è composta da un unico componente eseguibile, 
che dopo la costruzione è presente nella cartella **dist** dell'applicazione. 

## Ambiente di esecuzione 

L'applicazione **ciao-mondo** può essere eseguita nell'ambiente 
[standalone](../../environments/standalone/), 
sul nodo **node**. 

## Esecuzione 

Per eseguire l'applicazione **ciao-mondo**: 

1. posizionarsi nella cartella **dist** dell'applicazione 

2. eseguire lo script `../run-ciao-mondo.sh` 

In alternativa: 

1. posizionarsi nella cartella principale dell'applicazione 

2. usare il comando `gradle run` oppure `gradle run ciao-mondo`
