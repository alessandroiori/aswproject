#Snapcook ASW Project

Prerequisiti software:
* git
* Vagrant

Clonare la repository:

`$ git clone http://  snapcook`

Avviare ambiente:

`$ cd snapcook`

`$ vagrant up`

Avviare l'applicazione da browser tramite link:

http://localhost:8080/

Tecnologie utilizzate:
* Mongodb v3.0
* NodeJS v
* npm v
* expres v
* AngularJS v
* Chef
